# かけいぼ - iPhone向けお金管理アプリ

## 概要

iPhone向けに最適化したSPA（シングルページアプリケーション）形式のお金管理アプリです。手動入力による収支管理、口座残高管理、カテゴリー管理、月次グラフレポート、財布チェック差額自動調整、移動記録、iCloudバックアップ（エクスポート/インポート）を備えています。

---

## 実装済み機能

### ① ダッシュボード
- 月ナビゲーション（前月・次月切替）
- 月次収入/支出/収支差額サマリー
- 口座残高一覧（初期残高 + 取引の積み上げで自動計算）
- 最近の取引5件表示
- 財布チェック通知バナー（週1回の通知）

### ② 取引一覧
- 月別フィルタリング
- 種類・口座・カテゴリーによる絞り込みフィルター
- 日付グループで表示

### ③ 収支入力
- **収入 / 支出 / 移動** の3タイプ入力
- 金額・日付・カテゴリー・口座・メモの入力
- **移動記録の同時追加**（出発地/目的地/交通手段/目的）
- 既存取引の編集・削除

### ④ 口座管理
- 口座の追加・編集・削除（ソフトデリート）
- 種類: 現金 / 銀行 / クレジット / 電子マネー / その他
- 残高変更時に雑益/雑損取引を自動計上
- 財布チェック対象フラグの設定

### ⑤ カテゴリー管理
- 支出/収入カテゴリーの追加・編集・削除
- アイコン（絵文字）・カラー設定
- システムカテゴリー（削除不可）とユーザーカテゴリー

**システムカテゴリー（初期設定）:**
- 支出: 食費/交通費/日用品/医療費/娯楽/衣類/光熱費/通信費/雑費/**雑損**
- 収入: 給与/ボーナス/副業/**雑益**/贈り物

### ⑥ レポート
- 月別収支推移 **棒グラフ**（過去6ヶ月）
- カテゴリー別支出内訳（バー + パーセンテージ）
- 財布チェック履歴一覧

### ⑦ 財布チェック（差額自動調整）
- 対象口座のアプリ内計算残高と実際の残高を比較
- 差額がある場合 → **雑損（不足）/ 雑益（余剰）** として取引を自動生成
- チェック履歴を保存
- 週1回のリマインダーバナー表示

### ⑧ 移動記録
- 月別フィルタリング表示
- 出発地・目的地・交通手段・目的・金額を記録
- 収支入力と連動

### ⑨ iCloudバックアップ
- **エクスポート**: 全データをJSONファイルとしてダウンロード
- **インポート**: JSONファイルから復元（重複スキップ）
- iCloud Driveへの手動保存を推奨

## データモデル

| テーブル | 説明 |
|---------|------|
| `transactions` | 全取引（収入/支出/移動/調整） |
| `accounts` | 口座情報と種類 |
| `categories` | カテゴリー（アイコン・カラー付き） |
| `wallet_checks` | 財布チェック履歴 |
| `travel_logs` | 移動記録 |

### transactions フィールド
| フィールド | 型 | 説明 |
|-----------|-----|-----|
| id | text | UUID |
| type | text | income / expense / transfer |
| amount | number | 金額（正の整数） |
| account_id | text | 口座ID |
| to_account_id | text | 移動先口座ID（transferのみ） |
| category_id | text | カテゴリーID |
| date | text | YYYY-MM-DD |
| memo | text | メモ |
| is_adjustment | bool | 財布チェック差額調整フラグ |

- **HTML5 / CSS3 / Vanilla JavaScript（ES2022）**

- **Font Awesome 6.4** - アイコン
